# Conditionals

Scripts demonstrating `if/elif/else` logic, `for`/`while` loops, and classic programming problems.

## Scripts

| File | Description |
|------|-------------|
| `FizzBuzz.py` | Classic FizzBuzz — prints Fuzz, Buzz, or FizzBuzz for multiples of 3, 5, or both up to 50 |
| `celtofarein.py` | Temperature converter between Celsius and Fahrenheit based on user input (e.g. `30C`, `98F`) |
| `datatype.py` | Iterates a mixed list and prints the Python type of each element using `type()` |
| `divisible.py` | Finds all numbers between 1500–2700 divisible by both 7 and 5 |
| `even.py` | Prints all even numbers from 0 up to a user-specified value |
| `evenorodd.py` | Counts total odd and even numbers from 1 to n |
| `exlude0and3.py` | Loops through a range and skips 0 and 3 using `continue` |
| `factorial.py` | Computes factorial of n using a `for` loop |
| `fibinocci.py` | Generates first 10 Fibonacci numbers using tuple unpacking |
| `guess.py` | Number guessing game — keeps prompting until user enters a single-digit number (0–9) |
| `reverse.py` | Reverses a string entered by the user using index traversal |
| `vowels.py` | Counts total vowels and consonants in a user-entered string |

## Example

```bash
python3 FizzBuzz.py
python3 celtofarein.py   # enter: 100C or 212F
python3 factorial.py     # enter: 5
python3 vowels.py        # enter: Hello World
```
