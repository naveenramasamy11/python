# Basics

Core Python language concepts — loops, lists, and comparison operators.

## Scripts

| File | Description |
|------|-------------|
| `bool.py` | `while` loop countdown from 5 to 1 using a decrement condition |
| `list.py` | List operations: create, change, `len()`, `append()`, `insert()`, `remove()` |
| `relational.py` | Relational operators — compares user input against a fixed value using `<`, `>`, `==` |
| `while.py` | `while True` loop that reads user input and breaks when divisible by 10 |

## Example

```bash
python3 list.py
python3 relational.py   # enter a number when prompted
python3 while.py        # enter multiples of 10 to exit
```
