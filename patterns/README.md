# Patterns

Star and pyramid pattern printing scripts using nested loops.

## Scripts

| File | Description |
|------|-------------|
| `pattern.py` | Prints an hourglass star pattern — expanding then shrinking rows of `*` |
| `pattern1.py` | Right-aligned pyramid — takes user input for number of lines, prints centered `*` triangle |
| `pattern2.py` | Right-aligned growing star pattern — each row adds one `*` |
| `pyramid.py` | Full diamond pyramid function — prints both the upper and lower halves |
| `pyramid1.py` | Simple pyramid function — takes range as input, prints spaced `*` rows up and down |

## Example Output

`pattern.py`:
```
*
* *
* * *
* * *
* *
*
```

`pyramid1.py` (input: 5):
```

    *
   * *
  * * *
 * * * *
* * * * *
```

## Usage

```bash
python3 pattern.py
python3 pattern1.py   # enter number of lines when prompted
python3 pyramid.py    # prints a fixed 10-row diamond
python3 pyramid1.py   # enter range when prompted
```
