# Python Learning Repository

A structured collection of Python scripts organized by topic — from core language basics to object-oriented programming, system scripting, and error handling. Each folder includes examples with clear comments designed for beginners.

## Topics Covered

| Topic | Folder | Examples | Description |
|-------|--------|----------|-------------|
| Basics | [basics/](./basics) | 4 files | Variables, loops, lists, relational operators |
| Conditionals | [conditional/](./conditional) | 12 files | if/elif/else logic, loops, math problems, string operations |
| Functions | [functions/](./functions) | 8 files | Reusable functions: calculator, palindrome, sorting |
| Classes | [classes/](./classes) | 5 files | Object-Oriented Programming with classes and methods |
| Patterns | [patterns/](./patterns) | 5 files | Star and pyramid pattern printing using loops |
| System | [system/](./system) | 1 file | Shell/subprocess integration with Python |
| Error Handling | [error_handling/](./error_handling) | 2 files | try/except, multiple exceptions, custom exceptions |

## Requirements

- Python 3.7+

## Usage

Run any script directly:

```bash
python3 basics/list.py
python3 conditional/FizzBuzz.py
python3 functions/palindrome.py
python3 error_handling/try_except_basic.py
```

## Learning Path

Follow the topics in order for a structured learning experience:

```
basics/ → conditional/ → functions/ → classes/ → patterns/ → system/ → error_handling/
```
