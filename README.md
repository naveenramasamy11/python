# Python Learning Repository

A structured collection of Python scripts organized by topic — from core language basics to object-oriented programming and system scripting.

## Repository Structure

| Folder | Description |
|--------|-------------|
| [`basics/`](./basics) | Core Python concepts: loops, lists, relational operators |
| [`conditional/`](./conditional) | if/elif/else logic, loops, math problems, string operations |
| [`functions/`](./functions) | Reusable function examples: calculator, palindrome, sorting |
| [`classes/`](./classes) | Object-Oriented Programming with classes and methods |
| [`patterns/`](./patterns) | Star and pyramid pattern printing using loops |
| [`system/`](./system) | Shell/subprocess integration with Python |

## Requirements

- Python 3.7+

## Usage

Run any script directly:

```bash
python3 basics/list.py
python3 conditional/FizzBuzz.py
python3 functions/palindrome.py
```
