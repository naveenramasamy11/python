# Functions

Reusable Python functions covering math utilities, string operations, and list manipulation.

## Scripts

| File | Description |
|------|-------------|
| `calc.py` | Basic calculator function — performs add, subtract, divide, multiply with type validation |
| `getonlyeven.py` | Filters and returns only even numbers from a given list |
| `importarray.py` | Demonstrates use of Python's built-in `array` module to create typed arrays |
| `max3numbers.py` | Returns the maximum of three numbers passed as arguments |
| `palindrome.py` | Checks if a string is a palindrome by comparing first and second halves |
| `reverse.py` | Reverses a string using index traversal inside a function |
| `squareofnumbers.py` | Generates a list of squares for numbers 1 through n |
| `uniquelist.py` | Removes duplicate values from a list while preserving order |

## Example

```bash
python3 calc.py
python3 palindrome.py       # tests 'madam' and 'naveen'
python3 squareofnumbers.py  # squares of 1–10
python3 uniquelist.py       # deduplicates a sample list
```
