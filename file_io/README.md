# File I/O

> Read from and write to files using Python's built-in file handling functions.

## What You'll Learn

File I/O (Input/Output) is a fundamental skill for any developer. Python provides clean, readable syntax for working with text files, CSV data, JSON documents, and directories. Mastering file handling lets you build scripts that persist data, process logs, automate reports, and interact with the file system.

## Examples

| File | Description | Key Concepts |
|------|-------------|--------------|
| [read_write_text.py](read_write_text.py) | Reading and writing text files | `open()`, `with` statement, `read()`, `readlines()`, `write()`, `writelines()`, file modes |

## How to Run

```bash
python read_write_text.py
```

## Prerequisites

- Python 3.7+
- No external libraries required
