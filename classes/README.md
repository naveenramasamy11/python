# Classes

Object-Oriented Programming examples using Python classes, `__init__`, and instance methods.

## Scripts

| File | Description |
|------|-------------|
| `fibonacci.py` | `Fibonacci` class — takes two seed numbers and generates n terms of the sequence |
| `pyramid.py` | `Pyramid` class — renders a configurable pattern using range, spaces, and characters |
| `rectangular.py` | `Rectangle` class — calculates area given length and breadth via a `calc()` method |
| `reverse.py` | `Reverse` class — reverses a string passed at instantiation using index traversal |
| `student.py` | `Student` class — stores first/last name and computes a 25% salary increment |

## Concepts Covered

- Class definition and `__init__` constructor
- Instance variables (`self`)
- Instance methods
- Object instantiation and method calls

## Example

```bash
python3 rectangular.py   # Area of Rectangle: 12
python3 student.py       # Salary increments for multiple students
python3 fibonacci.py     # First 10 Fibonacci numbers via class
```
